Getting Started

Tag Name Description
<!Doctype html> Html announcement You start every html file with this, it lets the browser know the type of code you're using.
<head> Head The head of the page, the title, link to the stylesheet and any java based script will probably go here.
<title> Title Identifies the title of the page, this will appear in the tab name in your browser.
<body> Body The Body of the webpage. This is what will be displayed in the browser.
<Header, Main, Footer, Div> Different Division elements These are all used to seperate different areas of the page and organise your code.

Tables

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.
